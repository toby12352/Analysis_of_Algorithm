There's two files in the zip file. One is for insert sort and One is for merge sort. 

To run insert sort file
1)Type "g++ insertsort.cpp -o insert" to compile the file
2)Run by typing "./insert data.txt" to see the program

To run merge sort file
1)Type "g++ mergesort.cpp -o merge" to compile the file
2)Run by typing "./merge data.txt" to see the program
